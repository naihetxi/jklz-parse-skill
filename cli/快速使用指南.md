### 基本用法

```bash
cd /Users/lizhi/Desktop/金科览智/金科览智技能/jklz-parse-skill

# 1. 解析为 Markdown（默认）
python3 cli/jklz-parse.py parse 文档.pdf

# 2. 解析为 HTML
python3 cli/jklz-parse.py parse 文档.pdf --return html

# 3. 提取表格（JSON 格式）
python3 cli/jklz-parse.py parse 数据.xlsx --return table

# 4. 提取目录
python3 cli/jklz-parse.py parse 文档.pdf --return toc

# 5. 组合多种格式
python3 cli/jklz-parse.py parse 文档.pdf --return content#toc#table

# 6. 保存到文件
python3 cli/jklz-parse.py parse 文档.pdf -o 输出.md

# 7. 选择页面范围
python3 cli/jklz-parse.py parse 大文件.pdf --page-range "1-5,10"

# 8. 文档切片（用于 RAG 知识库）
python3 cli/jklz-parse.py parse 文档.pdf --return slice -o chunks.json

# 9. 健康检查
python3 cli/jklz-parse.py health

# 10. 查看配置
python3 cli/jklz-parse.py config --show
```

## 支持的格式

### 输入格式
- PDF (`.pdf`)
- Word (`.doc`, `.docx`)
- Excel (`.xlsx`)
- PowerPoint (`.ppt`, `.pptx`)
- 文本 (`.txt`)
- 图片 (`.png`, `.jpg`, `.jpeg`)

### 输出格式（--return 参数）

| 格式 | 说明 | 示例 |
|------|------|------|
| `content` | Markdown 文本（默认） | 适合阅读和编辑 |
| `html` | HTML 格式 | 适合网页展示 |
| `toc` | 目录结构（JSON） | 文档导航 |
| `table` | 表格数据（JSON） | 数据分析 |
| `slice` | 文档切片（JSON） | RAG 知识库 |

可以用 `#` 组合多个格式，如 `content#toc#table`

## 性能选项

```bash
# 高精度模式（默认 vl，适合表格密集文档）
python3 cli/jklz-parse.py parse 文档.pdf --image-mode vl

# 高性能模式（cv，速度更快）
python3 cli/jklz-parse.py parse 文档.pdf --image-mode cv
```

## 故障排除

### 1. 解析卡住
- 检查 API 服务状态：`python3 cli/jklz-parse.py health`
- 切换到高性能模式：`--image-mode cv`

### 2. 网络连接失败
- 确认内网可访问：`curl http://192.168.42.15:15216/metrics`
- 检查 VPN 或代理设置

### 3. API Key 错误
- 重新配置：`python3 cli/jklz-parse.py config --api-key YOUR_KEY`

## 高级用法

### 批量处理

```bash
# 批量解析 PDF
for file in *.pdf; do
  python3 cli/jklz-parse.py parse "$file" -o "${file%.pdf}.md"
done

```
